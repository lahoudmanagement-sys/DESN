# DESN

**Meteor Dynamics · Chicxulub** — a pitch deck for a morphing aircraft wing, designed for iPad display at a table presentation.

## What's in here

- `index.html` — single-file pitch deck (HTML + CSS + JS + SVG, all inline)
- `hero-bg.png` — hero background image

## Run locally

```
npx serve .
```

Then open `http://localhost:3000/`.

## Deploy

This is a static site — drop the files onto Vercel, Netlify, or GitHub Pages.
